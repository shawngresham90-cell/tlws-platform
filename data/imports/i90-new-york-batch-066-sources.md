# Batch 66 — I-90 New York (NY Thruway + Berkshire Connector: PA line/Ripley → Buffalo → Rochester → Syracuse → Utica → Amsterdam → Albany → MA line/Canaan): Source Report

Every listing was web-verified during research on **2026-07-15**. No field was invented: anything a source did not state is left blank in the CSV. Coordinates are blank on every row (geocoding is a separate verified workflow). Research method: web search against official brand/operator locators (Love's, Pilot/Flying J, TA/Petro, Sinclair, CAT Scale, state DOT/port-of-entry) as primary sources, with directory/review sites (iExit, Allstays, TruckMap, Find Truck Service, truckstopsandservices.com, coopsareopen.com, state rest-area guides) as secondary confirmation.

- Longest remaining I-90 state (~385 mi). The NY Thruway (I-90) is a ticketed tollway; its on-highway stops are the Applegreen-operated SERVICE AREAS (Angola, Pembroke, Ontario, Clifton Springs, Junius Ponds, Port Byron, Warners, DeWitt, Chittenango, Oneida, Indian Castle, Iroquois, Mohawk, Pattersonville, Guilderland) captured under 'parking'; off-Thruway truck stops (Love's, Pilot, TA, Flying J, Onvo, Kwik Fill) sit at numbered interchanges. Both included. Berkshire Connector (I-90 east of Albany) uses Bx exit numbers (Love's Canaan Exit B3).
- Cross-segment reconciliation: the Iroquois Service Area (MP 210, WB) was returned by both Central and East — kept once (East, which also carries its EB pair Indian Castle). McCarthy Tire appears in both West (Buffalo) and Central (East Syracuse) — distinct locations, both kept.
- CAT scales verified corridor-wide: Love's Ripley #833 (West), Love's Waterloo #820 + Pilot Liverpool #380 (Central), TA + Pilot Fultonville + Love's Canaan (East) — six distinct, each kept once.
- Service areas use milepost identifiers (mileposts DECREASE eastbound: Ontario 376 → Iroquois 210); kept in exit_number as 'MP nnn xB'. Angola is a single bidirectional plaza (listed once). NYSDOT Schodack is an inspection/weigh + rest area east of Albany (restrooms closed).
- Omitted, not fabricated: NY Thruway has NO fixed permanent weigh stations (NYSP/CVEU run portable inspections at service/rest areas) — only the NYSDOT Schodack WIM inspection site listed; the 'Blue Beacon Exit 35' in some summaries is Erie PA, not NY; Batavia-area hotels not confirmed to allow truck parking; STTC Syracuse street address left blank (conflicting listings).
