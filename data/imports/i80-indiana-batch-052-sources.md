# Batch 52 — I-80 Indiana (I-80/94 Borman Expressway: Hammond/Gary → Lake Station split; then I-80/90 Indiana Toll Road: South Bend → Elkhart → Ohio line): Source Report

Every listing was web-verified during research on **2026-07-15**. No field was invented: anything a source did not state is left blank in the CSV. Coordinates are blank on every row (geocoding is a separate verified workflow). Research method: web search against official brand/operator locators (Love's, Pilot/Flying J, TA/Petro, Sinclair, CAT Scale, state DOT/port-of-entry) as primary sources, with directory/review sites (iExit, Allstays, TruckMap, Find Truck Service, truckstopsandservices.com, coopsareopen.com, state rest-area guides) as secondary confirmation.

- Indiana already has 99 production rows (all on I-65); this batch was compiled against all 99 live importDupKey keys with 0 collisions.
- I-80 in Indiana never runs on its own alignment: it is concurrent with I-94 (the Borman Expressway) across the northwest (Hammond→Lake Station) and with I-90 (the Indiana Toll Road) across the north (Lake Station→Ohio line). All rows are on those shared corridors; interstate recorded as I-80. Toll-road service plazas use mile-marker labels in exit_number (mainline access, no numbered exit).
- Cross-segment reconciliation at the Lake Station split: the Blue Beacon Lake Station truck wash (returned by both segment researchers) is included once (Borman segment).
- Omitted, not fabricated: Love's Elkhart and Gallop's Goshen (both on the US-20 bypass, not the toll road); no verifiable state weigh station on the toll-road concurrency (electronic tolling; INDOT scales are on the free interstates).
