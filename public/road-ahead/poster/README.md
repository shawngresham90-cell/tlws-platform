# THE ROAD AHEAD — scene posters (drop-in folder)

One still per scene, named `scene-01.jpg` … `scene-07.jpg`. Shown before/behind
the clip, on slow or data-saving connections, and under reduced-motion.

- **Format:** JPG (or WebP), 1920×1080, **≤ 200 KB**.
- Easiest source: the clip's first frame. `scripts/compress-road-ahead-video.mjs`
  generates the matching poster automatically when you compress a clip.
